### 배운것

##### - java api

- `subString(시작인덱스)`: 문자열 시작인덱스부터 끝까지의 값을 반환한다.

##### - 커스텀 구분자 제대로 동작하지 않는 오류 해결하기

    커스텀 구분자 구분표시 \n를
    "\n"로 표시했을 때 테스트는 통과하지만 실제 구현에서는 실패한다.
    "\\n"로 표시했을 때 테스트는 실패하지만 실제 구현은 통과한다.

- 실제 구현 코드 -> `String endOfCustomDelimiter = "\\n";`
